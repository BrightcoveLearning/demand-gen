Managing Account Settings

To access the Demand Gen account settings, click the gear icon () in the navigation header.

Note: You must be an account administrator to access the ADMIN menu.

The account settings menu provides the following options:

  • Manage Users
  • Account Information
  • Analytics Settings
  • Billing
  • Google Analytics
  • Adobe Analytics

Manage Users

The Manage Users option allows you to manage the users who are authorized to use the Demand Gen account. For more information on managing users, see Managing Users.

Account Information

The Account Information option allows you to edit the account name and mailing address. For more information, see Managing Account Information.

Analytics Settings

The Analytics Settings allow you to set the default reporting time zone for the account. The reporting time zone is used to calculate day boundaries for reporting data. Changing the reporting time zone setting only affects data going forward, and changes are not applied retroactively. If you change this setting, you may see a flat spot or a spike in your analytics data when looking at day boundaries where the change was applied. Updates to this setting may not take effect immediately and analytics data may continue to refer to the old setting until the system processes the changes.

Billing

The Billing option allows you to view billing information for the account. For more information on billing, see Viewing Billing Information.

Google Analytics

The Google Analytics option allows you to configure Google Analytics integration for the account. For more information, see Configuring Google Analytics Event Tracking.

Adobe Analytics

The Adobe Analytics option allows you to configure Adobe Analytics integration for the account. For more information, see Configuring Adobe Analytics Event Tracking.
